Operator Notes

这一页主要解决“Agent 到底由什么组成”。真正的编辑入口仍然分散在其他页面，因为 ccbot 当前把文件系统作为 source of truth。

Open config Open env Open scheduler
  • 想扩展 Agent 行为，优先新增或改造 skill / prompt，而不是把大量固定流程塞回 system prompt。
  • 想接外部系统，优先落到 MCP；想沉淀工作方法，优先落到 skill。
  • 想调整顶层运行策略，则改 `config.json` 和 workspace `.claude` 文件。
